Function onePunch

  • onePunch(...favorites: string[]): string

    Filters, sorts, and processes a list of favorite strings. Removes all occurrences of the letters 'a' and 'e', sorts the words, and joins them into a single string. If no valid strings are provided, returns 'Broken!'.

    Parameters

    • ...favorites: string[]

      A variable number of string arguments representing Chuck Norris' favorite things.

    Returns string

    • A processed string with the required modifications or 'Broken!' if no valid input is given.
    onePunch('Beard', 'Jeans', 'Hairbrush', 'Knuckleduster', 'Sand') = 'Brd Hirbrush Jns Knuckldustr Snd'
    onePunch('Sock', 'Beard', 'Vest', 'Lady', 'Sage') = 'Brd Ldy Sg Sock Vst'
